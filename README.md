# Hankyung-Automation
Hankyung-article crawling and summarization, save, and upload automation projects

### Used
- Python (Jupyter Notebook)
- Selenium (Crawling)
- OpenAI  (Summarize with gpt-3.5-turbo API)
- GoogleDrive API v3 (Automate uploads)

### Requirements
- python >= 3.10.7
- selenium >= 4.15.2
- webdriver-manager >= 4.0.1
